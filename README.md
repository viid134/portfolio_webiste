## About This Project

This is a fully responsive personal portfolio website built to showcase my journey as a Computer Science student at St. Paul's University, my technical skills, and my growing expertise in digital marketing. The site is designed to serve as a central hub for my professional brand, highlighting my projects, core competencies, and creative approach to solving real-world problems.

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/viid134/portfolio_webiste.git
   ```

2. Open the project folder in your preferred code editor.
3. Open `index.html` in your web browser to view the project.

## Technologies Used

- **HTML5** – Semantic structure and content
- **CSS3** – Custom properties, flexbox, grid, and responsive design
- **JavaScript** – Interactivity (hamburger menu, smooth scrolling)
- **Font Awesome** – Iconography

## Requirements Needed

To contribute or work with this project, you need:

- Basic programming knowledge of HTML, CSS, and JavaScript.
- Visual Studio Code (or any other code editor of your choice).
- A modern web browser for testing.

## Key Features

- Fully responsive layout (optimized for desktop, tablet, and mobile).
- Smooth navigation with a dynamic hamburger menu.
- Terminal-style hero component for a unique developer aesthetic.
- Modular sections covering:
  - Professional summary and profile
  - Technical & digital marketing skills
  - Project case studies (Portfolio Website, Digital Marketing Campaign, Library DB System)
  - Contact information
